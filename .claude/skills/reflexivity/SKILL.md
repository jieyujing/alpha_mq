# SKILL.md - Reflexivity & Asymmetry (ARDE) Framework

该技能旨在解决复杂系统中的核心矛盾：如何在充满偏见且不可预测的环境中，通过识别反馈回路捕捉极端收益，同时规避毁灭性风险。

## 1. 底层逻辑 (The First Principle)
承认人类认知的必然缺陷（易错性），利用感知与现实之间的“双向偏离”寻找杠杆点，通过结构化的非对称布局（低成本试错+高上限回报）在极端波动中获益。

## 2. 核心架构：ARDE 四步决策法
当面临决策或分析任务时，遵循以下流程：

### 第一步：识别统计域 (Domain Mapping)
- **平庸世界 (Mediocristan)**：增长线性、可预测（如传统零售）。
- **极端世界 (Extremistan)**：爆发性强、黑天鹅频发（如 AI、金融市场）。
- *行动*：确认是否处于极端世界，若是，启动本模型。

### 第二步：建立杠铃布局 (Barbell Setup)
- **防御端 (90%)**：极度保守，确保生存底线。不加杠杆，确保即便失败也不会出局。
- **进攻端 (10%)**：极度投机，进行多个低成本“试错”实验。
- *关键*：放弃预测，转而对结构进行布局。

### 第三步：寻找反身性支点 (Identifying Reflexivity)
- **反馈回路识别**：区分趋向平庸的“负反馈”与背离均衡的“正反馈”。
- **认知与操作功能**：观察参与者的偏见如何改变基本面，基本面又如何强化偏见。
- **支点判定**：当大众偏见开始导致现实发生质变时，即为决策支点。

### 第四步：非对称加码 (Asymmetric Scaling)
- **叙事证伪**：不寻找支持证据，而是寻找能推翻当前“主流叙事”的断裂点。
- **动态加码**：一旦发现反馈回路成型，将进攻端资金集中于该支点，博取指数级增长。

## 3. 核心准则 (Soul Rules)
- **模糊的正确胜过精确的错误**：在复杂系统中，“精确”是安全感的敌人。
- **偏见是能量来源**：偏见驱动现实质变，是波动的红利所在。
- **错误是决策燃料**：只要不出局，频繁的小额错误是校准方向的唯一途径。
- **底线思维**：时刻自省——“如果我彻底错了，我会出局吗？”

## 4. 关键术语
- **反脆弱 (Antifragility)**：从冲击和混乱中获益。
- **负向知识 (Negative Knowledge)**：知道“什么行不通”比知道“什么行得通”更可靠。
- **叙事偏差 (Narrative Bias)**：警惕为了连贯故事而忽略随机性的天性。
